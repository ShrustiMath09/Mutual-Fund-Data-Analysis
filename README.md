# Mutual-Fund-Data-Analysis
# 📊 Mutual Fund Data Analysis Dashboard

## 📌 Project Overview
This project analyzes mutual fund performance data to evaluate investment trends, risk levels, and returns across different Asset Management Companies (AMCs). The goal is to provide insights that help in making data-driven investment decisions.

## 🎯 Objectives
* Analyze performance of mutual funds based on AUM, CAGR, and returns
* Compare funds across different risk categories
* Identify top-performing funds and investment patterns
  
## 📊 Dataset Details
* 📁 Records: 1,000+ mutual funds
* 🏢 AMCs Covered: 15+
* 📌 Key Features:
  * Assets Under Management (AUM)
  * CAGR (Compound Annual Growth Rate)
  * Returns (1 Month, 1 Year, 3 Years)
  * Risk Levels
  * Fund Categories
    
## 🛠 Tools & Technologies
* Power BI (Dashboard & Visualization)
* SQL (Data Extraction & Analysis)
* Excel / CSV (Data Source)
  
## 🔧 Data Processing
* Cleaned dataset by handling missing values and inconsistencies
* Standardized numerical formats for accurate analysis
* Structured data for efficient querying and visualization

## 📈 Key Analysis Performed

* Evaluated fund performance using **CAGR and return metrics**
* Conducted **risk-return analysis** using composite ratings
* Compared mutual funds across different **risk categories and AMCs**
* Analyzed **category-wise distribution of AUM**

## 📊 Dashboard Features
* Interactive Power BI dashboard
* KPI tracking (Total AUM ₹6.5M, Fund Distribution)
* Category-wise and AMC-wise comparison
* Filters for detailed analysis

## 🔍 Key Insights
* Identified **top-performing mutual funds** based on returns and risk balance
* High-risk funds showed **higher return potential but increased volatility**
* Certain AMCs consistently performed better across multiple categories
* Category-wise analysis revealed **dominant investment segments**

## 🚀 Project Outcome
This project demonstrates the ability to:
* Analyze financial datasets
* Perform risk-return evaluation
* Build interactive dashboards for decision-making
